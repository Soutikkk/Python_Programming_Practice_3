# 🐍 Python Programming Practice 3

A collection of beginner-friendly Python projects and practice programs covering computer vision, data visualization, audio processing, image processing, and more.

This repository is part of my Python learning journey, where I build small projects to strengthen my programming skills and explore popular Python libraries.

---

## 📂 Projects Included

### 👁️ Biometric Iris Scanner Visualization
- Simulates an iris scanner using OpenCV.
- Draws concentric circles and radial scanning lines.

**Libraries Used**
- OpenCV
- NumPy

---

### 🎵 Image → Piano Music Generator
- Converts a grayscale image into a sequence of piano notes.
- Demonstrates basic image processing and music generation.

**Libraries Used**
- Pillow
- music21

---

### 📊 Data Visualization

#### 📈 Bar Chart
Simple bar chart using Matplotlib.

#### 📦 Box Plot
Visualizes data distribution and outliers.

#### 🎻 Violin Plot
Displays probability distribution of the data.

#### 🔥 Heatmap
Visualizes matrix values using `imshow()`.

**Libraries Used**
- Matplotlib
- NumPy

---

### 🎤 Voice Recorder
- Records audio from the microphone.
- Saves the recording as a `.wav` file.

**Libraries Used**
- sounddevice
- scipy

---

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/Soutikkk/Python_Programming_Practice_3.git
```

Move into the project folder

```bash
cd Python_Programming_Practice_3
```

Install the required packages

```bash
pip install opencv-python numpy matplotlib pillow music21 sounddevice scipy
```

---

## 🚀 Run a Project

Example:

```bash
python iris_scanner.py
```

or

```bash
python voice_recorder.py
```
